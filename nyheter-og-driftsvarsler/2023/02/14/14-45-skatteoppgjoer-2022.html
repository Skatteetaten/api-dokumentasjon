Støtte for inntektsår 2022 for skatteoppgjørstjenestene

14. februar 2023

Skatteoppgjørstjenestene er nå klare for ekstern test for inntektsår 2022.

Det gjelder følgene tjenester:

  • Avregning API
  • Beregnet Skatt API
  • Pensjonsgivende inntekt API
  • Tilgjengeligdata API, Skatteplikt API
  • Summert skattegrunnlag API
  • Spesifisert summert skattegrunnlag API

Endelig kodeliste for Summert skattegrunnlag og Spesifisert summert skattegrunnlag er ikke implementert i Test pt., men blir tilgjengelig i starten av mars. Testdata er tilrettelagt og vi oppfordrer til å benytte Tenor testdatasøk for å finne relevante testdata.

Første skatteoppgjør for 2022 vil være tilgjengelig i Produksjon fra 25. mars.
