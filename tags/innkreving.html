ELAN - Utleggsbegjæring API

ELAN (Utleggsbegjæring API) er en tjeneste for å sende utleggsbegjæringer til namsmannen gjennom Skatteetatens systemløsninger.

Innkrevingsoppdrag API

Tjenesten dekker tjenester for å opprette, endre og avskrive innkrevingsoppdrag. I tillegg finnes tjenester for å avstemme hva som er oversendt til API-ene, se mottaksstatus og å hente eventuelle valideringsfeil.
