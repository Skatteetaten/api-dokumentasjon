Fra 23.juni støtter følgende tjenester skatteoppgjørsdata for inntektsår 2018 i prod:
 
- Avregning API
- Beregnet skatt API
- Pensjonsgivende inntekt API
- Skatteplikt API
- Summert skattegrunnlag API
- Spesifisert summert skattegrunnlag API
