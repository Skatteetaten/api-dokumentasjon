Følgende tjenester støtter nå skatteoppgjør for inntektsår 2018 i test:
 
- Avregning API
- Beregnet skatt API
- Pensjonsgivende inntekt API
- Skatteplikt API
- Summert skattegrunnlag API
 
Spesifisert summert skattegrunnlag API får 2018-støtte i uke 14.
